### Ejemplo 2 (Usuario avanzado)
**Consulta**: "cómo configuro load balancing en nginx?"
**Resultados**: [search_documents]: "Configuración de upstream en nginx.conf: upstream backend { server 192.168.1.1; server 192.168.1.2; }"
**Respuesta**: Para configurar load balancing en nginx, necesitás definir un bloque upstream en tu nginx.conf. Acá va un ejemplo:

```nginx
upstream backend {
    server 192.168.1.1;
    server 192.168.1.2;
}

server {
    location / {
        proxy_pass http://backend;
    }
}
```
